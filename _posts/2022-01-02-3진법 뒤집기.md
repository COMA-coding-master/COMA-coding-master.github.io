### 풀이내용

``` python
def solution(n):
    chk = n
    ans = []
    while chk:
        ans.append(chk % 3)
        chk //= 3
    ans.reverse()
    answer = 0
    for i, v in enumerate(ans):
        answer += v * (3**i)
    return answer
```
