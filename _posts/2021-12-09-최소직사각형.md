### 풀이내용

``` python
def solution(size):
    max_x = 0
    max_y = 0
    for i in range(len(size)):
        local_max = max(size[i])
        if max_x < local_max:
            max_x = local_max
        if size[i][0] < size[i][1]:
            if max_y < size[i][0]:
                max_y = size[i][0]
        else:
            if max_y < size[i][1]:
                max_y = size[i][1]
    return max_x * max_y
```
