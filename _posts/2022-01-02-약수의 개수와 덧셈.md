### 풀이내용

``` python
def solution(left, right):
    answer = 0
    for i in range(left, right + 1):
        yak = 0
        for y in range(1, i + 1):
            if i % y == 0:
                yak += 1
        if yak % 2 == 0:
            answer += i
        else:
            answer -= i
    return answer
```
