### 풀이내용

``` python
def solution(sizes):
    ret_list = []
    for idx, size in enumerate(sizes):
        if size[0] < size[1]:
            ret_list.append([size[1], size[0]])
        else:
            ret_list.append(size)
    
    max1 = max([ret[0] for ret in ret_list])
    max2 = max([ret[1] for ret in ret_list])
    answer = max1 * max2
    return answer
```
