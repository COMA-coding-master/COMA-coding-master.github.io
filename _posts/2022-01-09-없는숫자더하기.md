```java
class Solution {
    public int solution(int[] numbers) {
        final int NUM = 9;
        int answer = NUM * (NUM+1) / 2;
        for (int i = 0; i < numbers.length; i++) {
            answer -= numbers[i];
        }
        return answer;
    }
}
```
