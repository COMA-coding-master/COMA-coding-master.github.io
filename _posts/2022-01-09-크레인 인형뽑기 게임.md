### 풀이내용

``` python
def solution(board, moves):
    score = 0
    dolls = []
    for idx, move in enumerate(moves):
        for heigth, width in enumerate(board):
            if width[move-1]!=0:
                if len(dolls)>0 and dolls[-1]==width[move-1]:
                    score+=1
                    dolls.pop()
                else:
                    dolls.append(width[move-1])
                board[heigth][move-1]=0
                break
    return score*2
```
